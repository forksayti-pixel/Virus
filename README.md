# Virus
Вирус который может сделать так чтоб ваш компьютер завис, можно проверять на себе, напишу обход в README

Как пользоватся: все легко, копируем наш код, ну или же просто копию репозитория, и вставляем в Python IDLE, все нужно делать на ОС Windows ведь на линуксе и IOS, все не будет работать, в первую очередь это пк червь.
Что делает вирус: в начале спамит вкладками с блокнотами, рикроллами, и много звуков ошибок виндовс(КОЛИЧЕСТВО ВКЛАДОК/ЗВУКОВ РЕГУЛИРУЕМОЕ!!!), после же начинает вставлять синий экран с паролем который вы ставите ЛИЧНО в коде, на данный момент код стоит: 1234, так же легко обойти по Cntrl+Alt+Del и тд
